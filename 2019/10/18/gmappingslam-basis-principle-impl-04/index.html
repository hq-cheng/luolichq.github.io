GMapping SLAM基础、原理与实现（四）：概率学基础知识、贝叶斯滤波器、以及粒子滤波器

主要是关于一般移动机器人中应用的概率学基础知识，贝叶斯滤波器原理以及粒子滤波器原理。

4.1 应用概率学知识解决机器人SLAM问题

SLAM问题本质上是状态估计问题，其核心问题就是由SLAM数学模型对移动机器人位姿进行估计，同时完成环境地图的构建。第三节得到的SLAM数学模型为：

xt=f(xt−1,ut,wt)x_t = f(x_{t-1}, u_t, w_t) x​t​​=f(x​t−1​​,u​t​​,w​t​​)

zt=h(xt,mt,vt)z_t = h(x_t, m_t, v_t) z​t​​=h(x​t​​,m​t​​,v​t​​)

  • 状态值：xtx_tx​t​​；
  • 观测值：ztz_tz​t​​；
  • 控制输入值（里程计相邻时刻之间的变化值）：utu_tu​t​​；
  • 环境地图值（地图中的路标点，或特征点）：mtm_tm​t​​；
  • 噪声：wtw_tw​t​​，vtv_tv​t​​；

先暂时不考虑环境地图构建问题，将状态值xtx_tx​t​​和mtm_tm​t​​看作是同一个状态变量，主要对移动机器人的位姿估计问题进行分析，分析的方法是将整个SLAM数学模型中所有的物理量看作随机变量，引入概率学知识，对状态估计问题进行求解。因此，SLAM数学模型简化为：

xt=f(xt−1,ut,wt)x_t = f(x_{t-1}, u_t, w_t) x​t​​=f(x​t−1​​,u​t​​,w​t​​)

zt=h(xt,vt)z_t = h(x_t, v_t) z​t​​=h(x​t​​,v​t​​)

概率论分为两大学派，频率学派和贝叶斯学派。频率学派主张经过无限次抽样后，事件发生的概率趋近于一个确定的数。而贝叶斯学派主张的建立关于未知参数的模型，在没有观测到样本之前，一切参数都是不确定的（认为参数是一个随机变量，呈现为某种概率分布，而不是一个确定的数）。同时使用观测的样本值来估计参数，将得到的参数带入模型中，使当前模型最佳地拟合观测到的数据。

在该节中概率，概率密度，概率密度函数都是可以用p(x)p(x)p(x)表示，三者是可互换的。例如，随机变量X值为x的概率：

p(X=x)=p(x)p(X=x) = p(x) p(X=x)=p(x)

这里主要应用贝叶斯学派的概率论知识，将整个SLAM数学模型中所有的物理量看作随机变量（参数），利用历史的里程计控制数据u1:tu_{1:t}u​1:t​​和激光雷达观测数据z1:tz_{1:t}z​1:t​​，对当前时刻机器人位姿xtx_tx​t​​进行状态估计，得到状态变量xtx_tx​t​​可能的概率分布（或者说最优的概率分布）。同时，假设状态变量xtx_tx​t​​是满足隐马尔可夫链模型，即现阶段的状态变量只与历史的状态变量相关，与未来无关。最后，机器人位姿估计问题可以描述为：

已知前一时刻状态变量的估计分布p(xt−1∣z1:t−1,u1:t−1)p(x_{t-1}|z_{1:t-1}, u_{1:t-1})p(x​t−1​​∣z​1:t−1​​,u​1:t−1​​)，去估计当前时刻状态变量的估计分布p(xt∣z1:t,u1:t)p(x_t|z_{1:t}, u_{1:t})p(x​t​​∣z​1:t​​,u​1:t​​)，在这个估计过程中还会用到：

  • 状态转移分布（移动机器人运动模型更新）：p(xt∣xt−1,z1:t−1,u1:t)p(x_t|x_{t-1}, z_{1:t-1}, u_{1:t})p(x​t​​∣x​t−1​​,z​1:t−1​​,u​1:t​​)；

  • 观测分布（激光雷达观测模型）：p(zt∣xt,z1:t−1,u1:t)p(z_t|x_t, z_{1:t-1}, u_{1:t})p(z​t​​∣x​t​​,z​1:t−1​​,u​1:t​​)；

先了解一些概率学的基础知识，然后贝叶斯滤波器和粒子滤波器相关内容中将会给出具体的估计过程。

4.2 概率学基础知识

如果XXX和YYY是相互独立的随机变量，则有：

p(x,y)=p(x)p(y)p(x,y) = p(x)p(y) p(x,y)=p(x)p(y)

p(x∣y)=p(x)p(x|y) = p(x) p(x∣y)=p(x)

p(y∣x)=p(y)p(y|x) = p(y) p(y∣x)=p(y)

如果随机变量YYY携带有XXX相关的信息，即XXX和YYY互相关，若已知变量YYY值为yyy的概率，则有条件概率：

p(x∣y)=p(x,y)p(y)p(x|y) = \frac{p(x,y)}{p(y)} p(x∣y)=​p(y)​​p(x,y)​​

→p(x,y)=p(x∣y)p(y)=p(y∣x)p(x)\to p(x,y) = p(x|y)p(y) = p(y|x)p(x) →p(x,y)=p(x∣y)p(y)=p(y∣x)p(x)

条件独立：

p(x,y∣z)=p(x∣z)p(y∣z)p(x,y|z) = p(x|z)p(y|z) p(x,y∣z)=p(x∣z)p(y∣z)

即知道Z=zZ=zZ=z事件后，XXX和YYY相互独立，则有：

p(x∣y,z)=p(x∣z)p(x|y,z) = p(x|z) p(x∣y,z)=p(x∣z)

p(y∣x,z)=p(y∣z)p(y|x,z) = p(y|z) p(y∣x,z)=p(y∣z)

条件独立变量和普通的独立变量是两个概念，不能互相推导：

p(x,y∣z)=p(x∣z)p(y∣z)⇏p(x,y)=p(x)p(y)p(x,y|z) = p(x|z)p(y|z) \nRightarrow p(x,y) = p(x)p(y) p(x,y∣z)=p(x∣z)p(y∣z)⇏p(x,y)=p(x)p(y)

p(x,y)=p(x)p(y)⇏p(x,y∣z)=p(x∣z)p(y∣z)p(x,y) = p(x)p(y) \nRightarrow p(x,y|z) = p(x|z)p(y|z) p(x,y)=p(x)p(y)⇏p(x,y∣z)=p(x∣z)p(y∣z)

全概率公式（连续形式和离散形式）：

p(x)=∫p(x∣y)p(y)dyp(x) = \int p(x|y)p(y)\, dy p(x)=∫p(x∣y)p(y)dy

p(x)=∑yp(x∣y)p(y)p(x) = \sum_{y} p(x|y)p(y) p(x)=​y​∑​​p(x∣y)p(y)

贝叶斯公式：

p(x∣y)=p(y∣x)p(x)p(y)=ηp(y∣x)p(x)p(x|y) = \frac{p(y|x)p(x)}{p(y)} = \eta p(y|x)p(x) p(x∣y)=​p(y)​​p(y∣x)p(x)​​=ηp(y∣x)p(x)

这里p(y)p(y)p(y)不依赖xxx求解，所以将贝叶斯公式简化为比例形式，η=1/p(y)\eta = 1/p(y)η=1/p(y)称作归一化常数。

贝叶斯公式解释：p(x)p(x)p(x)可以看作是在不知道观测事件YYY发生之前，直观得出的关于XXX的概率分布，称作先验分布，而p(y∣x)p(y|x)p(y∣x)则是我们在事件XXX发生之后（已经得到先验分布之后），观测事件YYY发生并得到观测数据对应的概率分布，最后在观测数据的基础之上对之前事件XXX直观给出的概率分布（先验分布）进行修正，最终得到事件XXX的后验分布p(x∣y)p(x|y)p(x∣y)，即经过调整之后的概率分布。

条件贝叶斯公式：

p(x∣y,z)=p(y∣x,z)p(x∣z)p(y∣z)=ηp(y∣x,z)p(x∣z)p(x|y,z) = \frac{p(y|x,z)p(x|z)}{p(y|z)} = \eta p(y|x,z)p(x|z) p(x∣y,z)=​p(y∣z)​​p(y∣x,z)p(x∣z)​​=ηp(y∣x,z)p(x∣z)

置信度（belief）：为真实状态的每一个可能的假设，分配一个概率（或概率密度值）

置信分布：建立在控制数据u1:tu_{1:t}u​1:t​​和观测数据z1:tz_{1:t}z​1:t​​基础之上的状态的后验概率分布

bel(xt)=p(xt∣z1:t,u1:t)bel(x_t) = p(x_t|z_{1:t}, u_{1:t}) bel(x​t​​)=p(x​t​​∣z​1:t​​,u​1:t​​)

要求解置信分布，一般分为两个步骤：

  • 预测：利用之前时刻的状态后验分布，在控制执行之后，观测计算之前，得到状态的一个预测的置信分布（其实就是先验概率分布）；

bel‾(xt)=p(xt∣z1:t,u1:t)\overline{bel}(x_t) = p(x_t|z_{1:t}, u_{1:t}) ​bel​​​(x​t​​)=p(x​t​​∣z​1:t​​,u​1:t​​)

仅综合前一时刻的状态后验分布，和当前时刻的控制数据，得到这个预测的置信分布，此时还未利用当前时刻的测量数据。

  • 更新（或校正）：在预测基础之上，计算状态的置信分布（后验概率分布）

bel(xt)=p(xt∣z1:t,u1:t)bel(x_t) = p(x_t|z_{1:t}, u_{1:t}) bel(x​t​​)=p(x​t​​∣z​1:t​​,u​1:t​​)

4.4 贝叶斯滤波器近似实现：粒子滤波器

粒子滤波器是贝叶斯滤波器的一种近似实现方式。贝叶斯滤波器中，先验分布bel‾(xt)\overline{bel}(x_t)​bel​​​(x​t​​)中存在积分符号，是一个很难计算的量，为了解决这个问题，引入粒子滤波器，采用蒙特卡洛思想，利用一系列随机样本的加权平均值来近似后验分布bel(xt)bel(x_t)bel(x​t​​)的概率密度函数，通过求和来近似积分操作，即在滤波过程中，需要用到概率的地方，一概对变量采样，以大量采样及其相应的权值来近似表示概率密度函数。

粒子：从一个目标分布p(x)p(x)p(x)中采样得到的一系列样本xt[1],xt[2],...,xt[M]x_t^{[1]}, x_t^{[2]}, ..., x_t^{[M]}x​t​[1]​​,x​t​[2]​​,...,x​t​[M]​​，即称作粒子；

粒子群（粒子集）：从目标分布p(x)p(x)p(x)中采样得到的粒子构成的集合。例如，粒子滤波器中，对当前时刻状态变量的后验分布进行采样，得到的样本集合即为粒子群。

Xt:=xt[1],xt[2],...,xt[M]\mathcal{X}_t := x_t^{[1]}, x_t^{[2]}, ..., x_t^{[M]} X​t​​:=x​t​[1]​​,x​t​[2]​​,...,x​t​[M]​​

一个粒子xt[m]x_t^{[m]}x​t​[m]​​即为t时刻概率分布中可能存在的一个状态的实例。同时，用采样得到的粒子群Xt\mathcal{X}_tX​t​​去近似状态变量在t时刻目标分布p(xt)p(x_t)p(x​t​​)，即状态变量xt[m]x_t^{[m]}x​t​[m]​​被包含在粒子群Xt\mathcal{X}_tX​t​​中的可能性（likelihood，似然）。如果目标分布是后验分布bet(xt)bet(x_t)bet(x​t​​)，这种近似关系可以表示为：

xt[m] ∼ p(xt∣z1:t,u1:t)x_t^{[m]} \ \sim \ p(x_t|z_{1:t}, u_{1:t}) x​t​[m]​​ ∼ p(x​t​​∣z​1:t​​,u​1:t​​)

理论上当粒子数趋于无穷时上式才会成立，但只要粒子数不是非常少，有限个粒子数（大于100个）带来的误差是可以忽略的。

4.4.1 蒙特卡洛采样

蒙特卡洛采样的思想就是从从目标分布p(x)p(x)p(x)中采样得到NNN个样本（粒子），用这些样本的平均值来代替积分，求解该目标分布的期望值：

E[f(x)]=∫abf(x)p(x)dx=˙f(x1)+f(x2)+...+f(xN)NE[f(x)] = \int_{a}^b f(x)p(x)\, dx \dot= \frac{f(x_1)+f(x_2)+...+f(x_N)}{N} E[f(x)]=∫​a​b​​f(x)p(x)dx​=​˙​​​N​​f(x​1​​)+f(x​2​​)+...+f(x​N​​)​​

在贝叶斯滤波器中，目标分布即为状态变量的后验分布bel(xt)=p(xt∣z1:t,u1:t)bel(x_t)=p(x_t|z_{1:t}, u_{1:t})bel(x​t​​)=p(x​t​​∣z​1:t​​,u​1:t​​)，蒙特卡诺采样就是用来解决后验分布bel(xt)bel(x_t)bel(x​t​​)中积分难的问题。若能从后验分布中采样得到NNN个样本，那么后验概率的计算变成：

那么重采样这样简单地筛选粒子就没有问题了吗？

4.4.4 低方差重采样

上面这个简单的粒子滤波器递推算法存在两个问题：

  • 重采样中权重大的粒子复制几个？各个权重大的粒子之间复制个数怎么分配？
  • 粒子耗散的问题，即多次递推运算之后，很多粒子的权重都变得很小，可以忽略了，只有少数粒子的权重比较大，状态空间中的有效粒子数较少。粒子的多样性就逐渐耗散，最终所有的粒子都来自同一个粒子或少数几个粒子的复制。

一般采用有效粒子数来衡量粒子权值的退化程度，这里给出有效粒子数NeffN_{eff}N​eff​​的近似计算公式：

Neff=N1+var(wt) =˙ 1∑i=1N(wt[i])N_{eff} = \frac{N}{1+var(w_t)} \ \dot= \ \frac{1}{\sum_{i=1}^N(w_t^{[i]})} N​eff​​=​1+var(w​t​​)​​N​​ ​=​˙​​ ​∑​i=1​N​​(w​t​[i]​​)​​1​​

当有效粒子数NeffN_{eff}N​eff​​很小时，表明权重的方差很大，即权重大与权重小之间的差距很大，权值退化很严重；当NeffN_{eff}N​eff​​值小于设定的阈值，此时就应该在重要性采样之后采用低方差重采样的方法，来降低粒子权重的方差。
当有效粒子数NeffN_{eff}N​eff​​很大时，表明权重的方差很小，即权重大与权重小之间的差距很小，权值退化不太严重，可以不进行重采样操作；

由前面4.4.2节重要性采样可知，粒子滤波器滤波结果（求解期望值）实际变成了对proposal分布进行采样并求解加权和的形式：

p(xt∣z1:t,u1:t)=˙∑i=1Nw~t[i]δ(xt−xt[i])p(x_t|z_{1:t}, u_{1:t}) \dot= \sum_{i=1}^N\tilde{w}_t^{[i]}\delta(x_t - x_t^{[i]}) p(x​t​​∣z​1:t​​,u​1:t​​)​=​˙​​​i=1​∑​N​​​w​~​​​t​[i]​​δ(x​t​​−x​t​[i]​​)

其中w~t\tilde{w}_t​w​~​​​t​​是归一化之后的权重，有：

w~t[i]=wt[i]N\tilde{w}_t^{[i]} = \frac{w_t^{[i]}}{N} ​w​~​​​t​[i]​​=​N​​w​t​[i]​​​​

由前面4.4.1节蒙特卡洛采样可知，如果后验分布可以采样的话，求解的期望值（滤波结果）应表示为：

p(xt∣z1:t,u1:t)=˙1N∑i=1Nδ(xt−xt[i])p(x_t|z_{1:t}, u_{1:t}) \dot= \frac{1}{N}\sum_{i=1}^N\delta(x_t - x_t^{[i]}) p(x​t​​∣z​1:t​​,u​1:t​​)​=​˙​​​N​​1​​​i=1​∑​N​​δ(x​t​​−x​t​[i]​​)

而我们低方差重采样之后，实际就是希望得到上面这个结果，即从proposal分布中采样得到的粒子群{x~t[i],w~t[i]}\{ \tilde{x}_t^{[i]}, \tilde{w}_t^{[i]} \}{​x​~​​​t​[i]​​,​w​~​​​t​[i]​​}经低方差重采样之后变为{xt[i],1/N}\{ x_t^{[i]}, 1/N \}{x​t​[i]​​,1/N}，此时所有粒子权重一样都为1/N1/N1/N。

低方差重采样的具体操作：

  • 计算NeffN_{eff}N​eff​​值，判断其是否小于设定阈值；
    • 大于阈值，不进行（低方差）重采样，权重直接按照wt=wt−1p(zt∣xt)w_t = w_{t-1}p(z_t|x_t)w​t​​=w​t−1​​p(z​t​​∣x​t​​)更新；
    • 小于阈值，采用低方差重采样，重采样之后所有粒子权重都更新为1/N1/N1/N。
  • 低方差重采样采取下面的策略对粒子群进行滤除或复制：
    • 计算粒子的权值累积量λi,i=1,2,...N\lambda_i, i=1,2,...Nλ​i​​,i=1,2,...N；
    • 生成N个在[0, 1]区间均匀分布的随机数ui,i=1,2,...Nu_i, i=1,2,...Nu​i​​,i=1,2,...N；
    • 对于每一个uiu_iu​i​​，寻找归一化权值累积量大于或等于uiu_iu​i​​的最小标号mmm，即λm−1<ui<λm\lambda_{m-1}<u_i<\lambda_{m}λ​m−1​​<u​i​​<λ​m​​。当uiu_iu​i​​落在区间[λm−1,λm][\lambda_{m-1}, \lambda_{m}][λ​m−1​​,λ​m​​]时，粒子xt[m]x_t^{[m]}x​t​[m]​​被复制一次。

例如，假设有3个粒子，在第k时刻的时候，他们的权重分别是[0.1,0.1,0.8][0.1, 0.1 ,0.8][0.1,0.1,0.8], 然后计算他们的概率累计量得到[0.1,0.2,1][0.1, 0.2, 1][0.1,0.2,1]，归一化之后为[0.08,0.154,0.77][0.08, 0.154, 0.77][0.08,0.154,0.77]。接着，我们用服从[0,1][0,1][0,1]之间的均匀分布随机采样3个值，假设为0.15，0.38和0.54。也就是说，第二个粒子复制一次，第三个粒子复制两次。

最后，标准的粒子滤波算法流程为：

  • t=0时刻，粒子集初始化，对于i=1,2,...Ni=1,2,...Ni=1,2,...N，由当前时刻的proposal分布生成采样粒子{x0[i]}i=1N\{ x_0^{[i]} \}_{i=1}^N{x​0​[i]​​}​i=1​N​​；
  • 对于t=1,2,...Nt=1,2,...Nt=1,2,...N，循环执行以下步骤：
    • 重要性采样，对于i=1,2,...Ni=1,2,...Ni=1,2,...N，由当前时刻的proposal分布生成采样粒子{xt[i]}i=1N\{ x_t^{[i]} \}_{i=1}^N{x​t​[i]​​}​i=1​N​​，计算粒子权值wt[i]w_t^{[i]}w​t​[i]​​，并进行归一化处理，得到粒子群{x~t[i],w~t[i]}i=1N\{ \tilde{x}_t^{[i]}, \tilde{w}_t^{[i]} \}_{i=1}^N{​x​~​​​t​[i]​​,​w​~​​​t​[i]​​}​i=1​N​​；
    • 重采样，粒子群{x~t[i],w~t[i]}\{ \tilde{x}_t^{[i]}, \tilde{w}_t^{[i]} \}{​x​~​​​t​[i]​​,​w​~​​​t​[i]​​}经低方差重采样之后变为{xt[i],1/N}\{ x_t^{[i]}, 1/N \}{x​t​[i]​​,1/N}
  • 输出滤波结果，也就是计算当前t时刻的状态估计值（期望值）：x~t=∑i=1Nw~t[i]x~t[i]\tilde{x}_t = \sum_{i=1}^N\tilde{w}_t^{[i]}\tilde{x}_t^{[i]}​x​~​​​t​​=∑​i=1​N​​​w​~​​​t​[i]​​​x​~​​​t​[i]​​

这里滤波器算法只是按照《概率机器人》一书中的思路进行的推导，实际GMapping SLAM算法还会对粒子滤波器进行更多的优化。
